## System Requirement
**OS X 10.7(Lion) or later / iOS 5 or later**

## Installation
### CocoaPods
`pod 'SECoreTextView'`

### Manual Install
1. `git clone git@github.com:kishikawakatsumi/SECoreTextView.git`
2. Copy files in `Lib` and `Resources` to your project.
3. Add `CoreText.framework` to your target.
